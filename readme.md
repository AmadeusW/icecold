# Ice Cold Beer

A virtual rendition of the *Ice Cold Beer* arcade

[Play now](https://amadeusw.github.io/icecold/prototype)

## Objective

* Insert the ball into the highlighted socket
* 🍻

## Controls

* ↕ W/S • up/down ↕ • *Classic*
* ↕ W/S • U/J ↕ • *Narrow*
* ↕ W/S • I/J ↕ • *Narrow*
* ↕ D/S • K/J ↕ • *VIM*
